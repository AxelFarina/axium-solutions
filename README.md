# Axium Solutions — Company Website

Marketing site for Axium Solutions: ERP implementation, custom software, systems integration, process automation, and AI solutions.

Built with **React + Vite**. Bilingual (EN/ES). Deployed to GitHub Pages via GitHub Actions on every push to `main`.

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

Output goes to `dist/`.

## Deployment

Pushing to `main` triggers `.github/workflows/deploy.yml`, which builds the site and publishes it to GitHub Pages.
